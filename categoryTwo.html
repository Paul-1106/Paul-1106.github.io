Code Review:

Constructor and Destructor

The constructor function is when you start running the program for this project.

Once you have close down or exit the terminal, then the destructor function will do its work by clearing
the allocated memory and shutting down the terminal.

Load Courses

The load course function is a must when it comes to importing a CSV file.

The function will print out that it fails to read the CSV file if there are no CSV file in the project's folder.

If it does find the CSV file, then it should print out 2 columns which is the name of the course and the courses id.

Once that is finish, then it will automatically close the CSV file.

Display Courses

All of the courses listed from the csv file will be printed from the displayCourse function.

Add Courses

This function will add a node (course information) to the CSV file.

The node will will be place in order, but by default, it is not an alphabet order for the ID of the course
since it is not a unique variable.

Search Courses

This function will help other developers to search for the ID of the course inside of the CSV file

Sort Courses

This function is where you want to print each rows from the CSV file to be in alphabetic order

This is useful when searching for a course from a large amount of courses in CSV file

Remove Courses

This function will remove one course each based on the selected course ID

However, if the csv file is empty, then it will only return null
