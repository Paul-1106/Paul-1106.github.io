Code Review:



As you can see, this is an image of creating a 3D Shape called a box. Here are different ways to modify a 3D Shape:


- Scale(line 893): the scale is where you will resize the shape smaller or bigger.

- Rotation(line 896-898): the rotation is when you are modifying of how you are facing the 3D Shape.

- Position(line 901): the position is that you are moving the shape to a certain spot.

- Color(line 911): where it says SetShaderColor() function, it means that you can modify
the color of the 3D shape depending on the rgb you have set.
There are other functions like SetShaderMaterial() and SetShaderTexture().

3D Texture

In order to add texture to 3D shapes, it will need to be installed into the code with an image.


- Each bReturn variables will grab an image into a texture.

- the "BindGLTextures()" function will bind all of the images into 3D shape texture once each variables are completed.

3D Lightning


There are two types of lightning functions that will create a full shape will lighning effection:

- The code snippet on the left shows different lightning effections which are ambientColor, ambientStrength, diffuseColor, specularColor, and shininess.

- The "tag" from the seventh line will connect this function to a specific 3D shape depending on the name.


- The code snippet on the right side shows that each light sources will covered, so an order to have a full display 3D shape with
light effections, it needs to have lightSources[0], lightSources[1], lightSources[2], and lightSources[3]. (If missed any, then a 3D shape will look like
it is cut in half).
